Coursera - Getting and Cleaning Data

Running the analysis, Run_analysis.R is responsible for demonstrating the ability to collect, work with, and clean a data.
Make sure to run by running the Run_analysis.R inside 'UCI_HAR_Dataset' folder.

The steps involved to run the analysis are

Fork and clone this repository
In the R terminal, set your current working directory to the project root
install.packages("plyr") if you have not already installed plyr
source('./Run_analysis.R')
